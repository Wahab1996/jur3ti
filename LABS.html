Clinical Laboratory Tests Organized by Condition

Hypertension (HTN)

Basic Tests

  • Serum Creatinine, eGFR
  • Sodium, Potassium, Chloride
  • Fasting Blood Glucose
  • Lipid Profile (Cholesterol, LDL, HDL, Triglycerides)
  • Urinalysis (Protein, Albumin/Creatinine Ratio)

Additional Tests

  • TSH (to exclude secondary hypertension)
  • Plasma Renin and Aldosterone levels
  • Uric Acid
  • ECG / Echocardiogram (if available)

Diabetes Mellitus (DM)

Basic Tests

  • Fasting Blood Glucose
  • HbA1c
  • Oral Glucose Tolerance Test (OGTT)
  • Urine Microalbumin / Albumin-Creatinine Ratio
  • Serum Creatinine, eGFR
  • Lipid Profile

Additional Tests

  • Liver Function Tests (ALT, AST, ALP, Bilirubin, Albumin)
  • Electrolytes (Na, K, Cl)
  • TSH (rule out hypothyroidism)

Chronic Kidney Disease (CKD)

Basic Tests

  • Serum Creatinine, eGFR
  • BUN (Urea Nitrogen)
  • Urinalysis (Protein, RBCs, Specific Gravity)
  • Urine Protein 24h / Microalbumin
  • Electrolytes: Na, K, Ca, Phosphate, Mg

Additional Tests

  • Parathyroid Hormone (PTH)
  • Vitamin D (Calciferol)
  • Iron Studies (Iron, Ferritin, TIBC, Transferrin)

Liver Diseases

Basic Tests

  • ALT (Alanine aminotransferase)
  • AST (Aspartate aminotransferase)
  • ALP (Alkaline phosphatase)
  • GGT (Gamma glutamyl transferase)
  • Bilirubin (Total and Direct)
  • Albumin, Total Protein

Additional Tests

  • Prothrombin Time (PT)
  • Amylase, Lipase (if pancreatitis suspected)
  • Hepatitis serology (HBsAg, Anti-HBc, Anti-HCV)

Thyroid Disorders

Basic Tests

  • TSH
  • Free T3
  • Free T4

Additional Tests

  • Anti-TPO Antibody
  • Anti-Thyroglobulin Antibody
  • Anti-TSH Receptor Antibody

Anemia

Basic Tests

  • CBC with Differential
  • Reticulocyte Count
  • Peripheral Smear

Additional Tests

  • Serum Iron, Ferritin, TIBC, Transferrin
  • Vitamin B12, Folate
  • Hemoglobin Electrophoresis
  • G6PD Screen

Rheumatologic / Autoimmune Diseases

Basic Tests

  • ESR, CRP
  • Rheumatoid Factor (RF)
  • Anti-CCP
  • ANA, Anti-dsDNA

Additional Tests

  • Complement C3, C4
  • Anti-Sm, Anti-RNP (if indicated)
  • Cardiolipin Antibodies (IgG, IgM)

Infectious Diseases

Common Panels

  • HIV 1+2 Antibodies
  • HBsAg, Anti-HBc, Anti-HBs
  • HCV Antibody
  • Syphilis (RPR)
  • TB Test

Microbiology Cultures

  • Urine Culture
  • Stool Culture
  • Throat Culture (Strep A)
  • Wound or Vaginal Swab Culture

Cardiac Evaluation

Basic Tests

  • Troponin I
  • CK, CK-MB
  • BNP (Natriuretic Peptide)
  • Lipid Profile

Reproductive & Hormonal Evaluation

Female

  • FSH, LH
  • Estradiol
  • Progesterone
  • Prolactin
  • β-hCG (Serum/Urine)

Male

  • Testosterone
  • DHEA
  • Prostate Specific Antigen (PSA)

General / Routine Screening

  • CBC
  • Lipid Profile
  • Blood Glucose (Fasting & Postprandial)
  • Liver Function Tests
  • Renal Function Tests
  • Urinalysis
  • Vitamin D, Vitamin B12
